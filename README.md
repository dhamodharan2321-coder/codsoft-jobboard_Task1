💼 Job Portal Website (Task 1)
🚀 This is a Job Portal Web Application developed as part of my internship with CodSoft.

📌 Project Overview
The Job Portal allows employers to post job openings and candidates to search and apply for jobs.
It simulates real-world job portal functionalities using frontend technologies.

✨ Features
🏠 Home Page with featured job listings
🔍 Job Listings Page with search functionality
📄 Job Details Page (dynamic content)
🏢 Employer Dashboard (post jobs)
👤 Candidate Dashboard (profile management)
📨 Job Application form with resume upload
🔐 Login & Registration system
🔒 Protected pages (login required)
📱 Fully responsive design
🛠️ Tech Stack
HTML5
CSS3
JavaScript
LocalStorage (for data handling)
🌐 Live Demo
👉(https://dhamodharan-jobboard.netlify.app/)

📂 Project Structure
Task-1-Job-Board │ ├── index.html ├── jobs.html ├── job-details.html ├── employer.html ├── candidate.html ├── apply.html ├── login.html ├── register.html │ ├── css/ │ └── style.css │ └── js/ └── script.js

🔐 Authentication Flow
User registers using the registration page
User logs in using credentials
Access to pages is restricted without login
Session is maintained using localStorage
